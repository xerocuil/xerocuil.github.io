**Usage**

Command     | Description
:------     | :----------
`git.c <msg>` | Commit with \<msg\> and timestamp.
`git.l <n>`   | Show last \<n\> commits (Default: 10)
`git.p`       | Push to origin
`git.qs`      | Quick save: Add, Commit, Push
`git.rm`      | Remove file from repository
